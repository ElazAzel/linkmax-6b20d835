# Business Zone Skill

Работа с CRM-модулем LinkMAX: квалификация лидов, анализ конкурентов, генерация доменов.

## Когда использовать

- Квалификация входящих лидов (New → Contacted → Won/Lost)
- Анализ рекламных креативов конкурентов
- Генерация доменных имён для страниц пользователей
- Поиск целевых компаний по ICP
- Подготовка outreach-стратегий

## Воркфлоу

### 1. Квалификация лида

При появлении нового лида (источник: Telegram-бот, форма на странице):

1. Извлечь данные: имя, контакт, услуга, источник
2. Определить скоринг (1-10) по ICP:
   - Совпадение по нише (Creator / Freelancer / Small Business)
   - Наличие бюджета (сигналы: запрос цены, корпоративный email)
   - Срочность (сигналы: "срочно", "нужно сегодня")
3. Установить статус в CRM (таблица `leads`):
   - Score 8-10 → `contacted` + Telegram-уведомление владельцу
   - Score 5-7 → `new` (ожидает обработки)
   - Score <5 → `lost` (авто-отписка "спасибо за интерес")
4. Создать запись в таблице `lead_history` с комментарием

**Ключевые файлы:**
- `src/components/zones/leads/` — компоненты CRM
- `src/repositories/lead.repository.ts` — работа с лидами
- `supabase/migrations/` — схема таблиц

### 2. Анализ конкурентов

По запросу пользователя:

1. Определить конкурента (название, ниша, регион)
2. Извлечь рекламу из Facebook Ad Library:
   - URL: `https://www.facebook.com/ads/library/?active_status=all&ad_type=all&country=ALL&q={COMPANY}`
3. Структурировать анализ:
   - Проблемы, которые поднимают
   - УТП и форматы креативов
   - CTA-паттерны
4. Сравнить с позиционированием LinkMAX

**Пример вывода:**
```markdown
# Анализ конкурента: Taplink

## Сообщения
- "Одна ссылка для всего" (4 объявления)
- CTA: "Попробуйте бесплатно"

## Рекомендация
У Taplink сильный фокус на простоте. LinkMAX стоит контрастировать
на AI-функциях и Business Zone (CRM + Kanban).
```

### 3. Генерация доменов

По запросу пользователя на создание страницы:

1. Извлечь ключевые слова из профиля пользователя
2. Сгенерировать 5-10 вариантов доменов:
   - `.my` (основной домен LinkMAX)
   - Вариации: `{username}.lnkmx.my`, `{brand}.bio`
3. Проверить доступность (whois-like запрос)
4. Предложить 3 лучших варианта с обоснованием

## Связанные модули

- `communications` — Telegram-уведомления о новых лидах
- `analytics` — PostHog-события по воронке лида
- `content-creation` — AI-генерация страниц для конверсии
