

# Настройка всех Telegram-уведомлений со страницы пользователя

## Обнаруженные проблемы

### Критические баги в существующих функциях

1. **`send-social-notification`** — СЛОМАНА. На строке 93 результат присваивается в `response`, но на строках 94-96 используется несуществующая переменная `result`. Функция всегда падает с ReferenceError.

2. **`send-zone-notification`** — СЛОМАНА. На строке 25 проверяется `telegramBotToken`, которая нигде не объявлена. Функция всегда возвращает 500 "Configuration missing".

3. **`send-team-notification`** — Некорректная проверка. `sendMessage()` возвращает parsed JSON (из `callTelegram`), а код на строке 80 проверяет `telegramResponse.ok` как будто это Response объект. Нужно проверять `telegramResponse.ok` из parsed Telegram API response (это корректно, т.к. Telegram API возвращает `{ok: true}`).

### Отсутствующие уведомления

4. **Newsletter подписка** — пользователь НЕ получает Telegram-уведомление когда кто-то подписывается на рассылку через его страницу. Лид создаётся, но `send-lead-notification` не вызывается.

5. **Лайки страницы** — пользователь НЕ получает уведомление когда кто-то лайкает его страницу в галерее.
