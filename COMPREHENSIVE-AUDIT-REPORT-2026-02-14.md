## 🚨 КРИТИЧЕСКИЕ ПРОБЛЕМЫ

### 1. Невалидный JSON в файле локализации (КРИТИЧНО)

**Файл:** `src/i18n/locales/kk.json`  
**Проблема:** JSON парсится с ошибкой `Unexpected token`  
**Симптомы:**
- i18n скрипт падает: `npm run i18n:check` не работает
- Казахский язык (KK) недоступен для пользователей
- Блокирует CI/CD процесс локализации

**Решение:**
```bash
# Попробовать восстановить файл
npm run i18n:import
# Или вручную отредактировать и переформатировать
node cleanup_json.js
```

**Рекомендация:** Немедленно исправить этот файл

### 3. Критические ошибки типизации (ВЫСОКИЙ ПРИОРИТЕТ)

**Файл:** [src/components/admin/AdminUsersTab.tsx](src/components/admin/AdminUsersTab.tsx#L129)  
**Ошибка:** Parameter 'v' implicitly has an 'any' type

```typescript
// ❌ Текущий код (строка 129)
<Select onValueChange={(v) => extendTrial.mutate({ userId: adminUser.id, days: parseInt(v) })}>

// ✅ Исправленный код
<Select onValueChange={(v: string) => extendTrial.mutate({ userId: adminUser.id, days: parseInt(v) })}>
```
