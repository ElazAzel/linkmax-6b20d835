## 2) Методика проверки

### 2.1 Freshness-проверка

Для воспроизводимости добавлен CLI-скрипт:

```bash
node scripts/audit-docs-freshness.mjs --scope=docs --days=60 --since=2026-04-18
```

Скрипт:
- собирает все markdown-файлы через `rg --files`;
- для каждого файла берёт дату последнего коммита (`git log -1 --format=%cs -- <file>`);
- рассчитывает возраст в днях;
- классифицирует файлы как fresh/stale относительно cutoff.
