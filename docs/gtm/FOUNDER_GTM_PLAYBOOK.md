# Founder-Led GTM Execution System: First 100 Users

## Daily Motion
Research → DM → Onboard → Publish → Verify bio → Follow up for booking → Collect proof.

**Main offer**: "Соберу вам страницу для записи клиентов за 10 минут. Бесплатно."
**Main channel**: Instagram DMs to beauty masters in Almaty/Astana.

---

## Target Segments

### Tier 1 — High-Fit (60% effort)
- Nail techs, lash artists, brow masters in Almaty/Astana
- Instagram 500-15K followers, post work regularly
- Currently "пишите в DM" or Taplink without booking
- Signal: highlight "Прайс"/"Запись" in stories but no link

### Tier 2 — Medium-Fit (30% effort)
- Barbers, makeup artists, cosmetologists
- Basic Taplink/linktree, no booking flow

### Skip
- Salons 5+ staff, coaches/realtors, <200 followers, already on paid booking platform

### Lead Scoring
| Signal | Points |
|--------|--------|
| "Запись в DM" in bio | +3 |
| Work portfolio in posts | +2 |
| 1K-10K followers | +2 |
| Uses Taplink/no-booking link | +2 |
| Almaty/Astana location | +1 |
| Posts stories daily | +1 |
| Already has booking tool | -5 |

## Outreach Messages

### Cold DM #1
> Привет! Увидела ваши работы — очень красиво 💅
> Делаю бесплатный сервис для мастеров: ссылка для записи клиентов прямо из Инстаграма.
> Вот пример как выглядит: [lnkmx.my/demo-nails]
> Могу собрать такую же под вас за 10 минут, бесплатно. Интересно?

### Follow-up #1 (через 2 дня)
> Привет! Написала пару дней назад про ссылку для записи.
> Вот как это работает у другого мастера: [screenshot/ссылка]
> Если интересно — напишите, соберу вашу за 10 мин 🙌

### Follow-up #2 (через 5 дней)
> Последний раз напоминаю 😊
> Если когда-нибудь надоест "запись в DM" — вот бесплатная альтернатива: lnkmx.my/for-masters
> Удачи в работе! 💅

### После ответа "да"
> Супер! Чтобы собрать страницу, пришлите:
> 1. Ваше имя / название
> 2. Фото для аватарки
> 3. Список услуг + цены
> 4. Контакт для связи (WhatsApp/Telegram)
> Через 10-15 минут покажу готовый результат 🔥

### После сборки
> Готово! Вот ваша страница: [lnkmx.my/slug]
> Осталось вставить ссылку в Инстаграм био — и клиенты смогут записываться сами.
> Как вставить: Профиль → Редактировать → Сайт → вставить ссылку.
> Сделаете? Могу помочь если что 🙌

### Через 3 дня после публикации
> Привет! Как страница — есть первые записи?
> Если хотите, могу подсказать как сделать первый пост/сторис со ссылкой, чтобы клиенты увидели.

### Запрос кейса (после первой записи)
> Круто что пошли записи! 🎉
> Можно использовать ваш пример (без личных данных) чтобы показать другим мастерам?
> Скрин записи или короткий отзыв — было бы супер 🙏

### Реактивация (через 2 недели)
> Привет! Мы обновили сервис — теперь клиент видит свободные слоты и может оплатить сразу.
> Ваша страница всё ещё активна: [url]
> Хотите обновлю что-нибудь?

## Concierge Onboarding Flow

| Step | Action | Time |
|------|--------|------|
| 1. Intake | Get: name, avatar, services+prices, phone/WA | 2 min |
| 2. Build | Create page + booking block + price list | 10 min |
| 3. Review | Send preview, get approval/tweaks | 5 min |
| 4. Publish | Publish, send final URL | 1 min |
| 5. Bio | Send "как вставить в био" instructions | 2 min |
| 6. Verify | Check Instagram bio 24h later | 1 min |
| 7. First traffic | Suggest story "Записаться можно по ссылке в био ☝️" | DM |
| 8. First booking | Check in 3 days, celebrate or troubleshoot | 2 min |

**~25 min per user over 1 week. 8-10 onboardings/week = 100 users in 5-7 weeks.**

## Funnel Metrics & Drop-off Fixes

| Stage | Target | Drop-off Fix |
|-------|--------|-------------|
| DM → Reply | >15% | Better opening, real example |
| Reply → Onboarding | >70% | Dead simple intake (4 items) |
| Onboarding → Publish | >90% | Build <15 min, same session |
| Publish → Bio | >60% | Screenshot instructions, verify next day |
| Bio → First traffic | >0 in 48h | Suggest story template |
| Traffic → First booking | >10% of published | Improve page, add prices |
| Booking → Case study | >30% | Ask right after first booking |
