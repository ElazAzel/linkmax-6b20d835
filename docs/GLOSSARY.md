# Глоссарий

Краткий справочник терминов проекта lnkmx (InkMax).

## Продуктовые и доменные термины

| Термин | Описание |
|--------|----------|
| **Lnkmx / LinkMax** | Платформа — Micro-Business OS: page builder + Mini-CRM + аналитика + финтех-основа. |
| **Solo-Economy** | Экономика независимых специалистов (креаторы, фрилансеры, микробизнес). |
| **Page / Страница** | Мини-сайт пользователя (до 6 страниц на Pro). |
| **Block / Блок** | Элемент контента в редакторе (ссылка, кнопка, бронирование, каталог и т.д.). Всего 28+ типов блоков. |
| **Mini-CRM** | Встроенный CRM: лиды, статусы, уведомления в Telegram. |
| **Business Zone** | Рабочие пространства: Kanban сделок, задачи, контакты, счета и документы. |
| **Invoices / Счета** | Выставление счетов внутри зон (multi-item, авто-нумерация, статусы). |
| **EDO / ЭДО** | Генератор документов: акты, договоры и счета на базе HTML-шаблонов. |
| **Liquid Glass** | Дизайн-система платформы: glassmorphism, микро-анимации. |
| **Pixel Proxy** | Серверная отправка событий в рекламные системы (FB CAPI, TikTok) для обхода блокировщиков. |

## Технические термины

| Термин | Описание |
|--------|----------|
| **Edge Functions** | Серверная логика на Supabase (Deno). 28+ функций. |
| **RLS** | Row Level Security — разграничение доступа к строкам БД на уровне PostgreSQL. |
| **SSR (для ботов)** | Pre-render HTML для поисковых роботов через Cloudflare Worker + Supabase Edge. |
| **AEO/GEO** | Answer Engine Optimization / Generative Engine Optimization — схемы и контент для AI-поиска. |
