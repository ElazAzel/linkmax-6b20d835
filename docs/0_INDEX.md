# lnkmx Product Documentation Base

This directory contains the core "Source of Truth" documents for the **lnkmx** platform. These 6 documents cover the entire business from Vision to Operations.

## The 6 Pillars

1.  [**Product Vision & Strategy**](./1_PRODUCT_VISION.md)
    *   *The "Why"*. Mission, Vision, Target Audience, and Strategic Pillars.
2.  [**Platform Architecture**](./2_PLATFORM_ARCHITECTURE.md)
    *   *The "How"*. Technical stack, system design, data models, and security.
3.  [**Marketing & GTM Strategy**](./3_MARKETING_STRATEGY.md)
    *   *The "Growth"*. User acquisition, pricing, brand identity, and launch plan.
4.  [**Investment Memo**](./4_INVESTMENT_MEMO.md)
    *   *The "Business Case"*. Market sizing (TAM), unit economics, and financial ask.
5.  [**Product Roadmap**](./5_PRODUCT_ROADMAP.md)
    *   *The "Timeline"*. Feature milestones for the next 12 months.
6.  [**Operational Handbook**](./6_OPERATIONAL_HANDBOOK.md)
    *   *The "Runbook"*. Team roles, support protocols, legal, and compliance.
