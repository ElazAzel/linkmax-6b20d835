# Документация lnkmx (InkMax)

Централизованная документация по платформе: архитектура, фичи, деплой, аудиты и операционные процедуры.

## С чего начать

| Кому | Документ |
|------|----------|
| **Новый разработчик** | [getting-started/DEVELOPER-QUICKSTART.md](getting-started/DEVELOPER-QUICKSTART.md) — установка, команды, структура проекта. |
| **Обзор продукта и архитектуры** | [architecture/COMPREHENSIVE_PLATFORM_GUIDE.md](architecture/COMPREHENSIVE_PLATFORM_GUIDE.md) — видение, модули, стек, блоки. |
| **Текущее состояние системы** | [audits/PLATFORM_SNAPSHOT.md](audits/PLATFORM_SNAPSHOT.md) — единый источник истины по платформе. |

## Навигация

- **[INDEX.md](INDEX.md)** — краткий индекс по разделам (архитектура, безопасность, деплой, фичи, аудиты).
- **[DOCUMENTATION-INDEX-FULL.md](DOCUMENTATION-INDEX-FULL.md)** — полный каталог документов с разбивкой по папкам и ролям.

## Структура папок

| Папка | Содержимое |
|-------|-------------|
| **getting-started/** | Онбординг, быстрый старт. |
| **architecture/** | Архитектура, стек, паттерны, производительность, граф зависимостей. |
| **audits/** | Аудиты, отчёты, PLATFORM_SNAPSHOT (см. [audits/README.md](audits/README.md)). |
| **deployment/** | Чек-лист деплоя, GitHub Actions, runbooks. |
| **seo/** | SEO, SSR для ботов, AEO/GEO (см. [seo/README.md](seo/README.md)). |
| **security/** | Безопасность, RLS. |
| **features/** | Фичи, UI-компоненты, блоки. |
| **implementation/** | API, БД, i18n, демо-аккаунты. |
| **product/** | Видение, маркетинг, инвестиции, roadmap. |
| **testing/** | Стратегия тестирования. |
| **operations/** | Операционное руководство, tech debt, a11y, changelog. |
| **ADR/** | Architecture Decision Records. |
| **presentation/** | Презентации. |
